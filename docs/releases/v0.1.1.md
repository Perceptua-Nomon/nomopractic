# nomopractic v0.1.1

**Released:** 2026-03-10
**Type:** Bug fix

---

## Summary

Patch release fixing two bugs in the ADC/battery subsystem discovered during
live integration testing at v0.1.0. Both bugs caused incorrect readings on real
hardware; the mocked tests at v0.1.0 did not catch them.

---

## Bug Fixes

### ADC command byte formula
- **Before:** `0x10 + channel`
- **After:** `0x10 | (7 - channel)`
- **Impact:** All ADC channels read incorrect data. The Robot HAT V4 register
  map sends channels in reverse order (A7=0x10, A0=0x17); the additive formula
  produced the right byte only for channel 0 by coincidence.

### Battery voltage scaling
- **Before:** `voltage_v = raw × 3`
- **After:** `voltage_v = (raw / 4095) × 3.3 × 3.0`
- **Impact:** Reported voltage was ~12 000× too large for typical ADC readings.
  Correct formula accounts for the 12-bit ADC range (4095 max), 3.3 V reference
  voltage, and the 3:1 resistor divider on the battery sense line.
